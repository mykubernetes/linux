
## POSIX正则表达式中的字符类

| 类 | 匹配字符 |
|----|----------|
| [[:alnum:]] | 文字，数字字符 |
| [[:alpha:]] | 字母字符 |
| [[:lower:]] | 小写字母 |
| [[:upper:]] | 大写字母 |
| [[:digit:]] | 小数 |
| [[:xdigit:]] | 十六进制数字 |
| [[:punct:]] | 标点符号 |
| [[:blank:]] | 制表符和空格 |
| [[:space:]] | 空格 |
| [[:cntrl:]] | 所有控制符 |
| [[:pring:]] | 所有可打印的字符 |
| [[:graph:]] | 除空格外所有可打印的字符 |

## 用于重复模式的元字符

| 元字符 | 描述 |
|--------|------|
| * | 重复0次或更多次 |
| + | 重复1次或更多次 |
| ？ | 重复0次或1次 |
| {n} | 重复n次 |
| {n,} | 重复n次或更多次 |
| {n,m} | 重复不少于n次，不多于m次 |
